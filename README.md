## 🚀 Quick Start

This project uses [uv](https://github.com/astral-sh/uv) for ultra-fast Python package and project management.

### 1. Prerequisites
Ensure you have `uv` installed on your system. If you're on Ubuntu/Linux:
```bash
curl -LsSf [https://astral.sh/uv/install.sh](https://astral.sh/uv/install.sh) | sh
```

### 2. Setup Repository
```bash
git clone [https://github.com/your-username/my-cool-project.git](https://github.com/your-username/my-cool-project.git)
cd my-cool-project
uv sync
```
